<h3> The Cure - Fanpage Project </h3>
<p>
Este foi o meu primeiro projeto grande de desenvolvimento web! É uma fanpage dedicada à banda britânica The Cure, criada com o objetivo de praticar os fundamentos do desenvolvimento front-end.
</p>

<h4> Sobre o Projeto </h4>
<p>
O site funciona como uma central de informações sobre a banda, apresentando uma estrutura clara e organizada para fãs e entusiastas. O foco principal foi aprender a estruturar documentos web de forma semântica.
</p>


Neste projeto você encontrará:

<ul>
<li> Resumo sobre a banda </li>
<li> Integrantes (Atuais e antigos) </li>
<li> Discografia </li>
</ul>

<h4> Tecnologias Utilizadas </h4>

HTML5: Estruturação de textos, listas, imagens e links.
<br>
CSS3: Estilização básica para cores, fontes e layout.
